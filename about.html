Why This Stack?

One API key. One repository. One deployment platform. Maximum intelligence.

🧠

Claude Does Everything

Resume parsing, adaptive MCQ generation, code review, HR simulation, behavioral scoring — all powered by a single Anthropic API key via direct fetch() calls from the browser. No middleware needed.

📦

GitHub as Backend

The GitHub repository is both the codebase and the CI/CD pipeline. GitHub Actions auto-builds and deploys every push. GitHub Pages serves the static frontend globally with HTTPS out of the box.

⚡

Zero Infrastructure

No FastAPI, no Docker, no cloud VMs, no databases. All state lives in localStorage and sessionStorage. All intelligence lives in Claude. The entire platform is a folder of HTML, CSS, and JS files.

🔐

API Key Security

The ANTHROPIC_API_KEY is stored as a GitHub Actions secret and injected at build time into a config file — never committed to the repo or exposed in public source. Requests go directly to api.anthropic.com.

Prompt Engineering — Module 1
System Prompt — Resume Parser
Role: Expert resume parser. Extract structured data. Return only valid JSON.
Schema: {name, email, phone, github, education[], experience[], skills[], projects[]}
Rules: No markdown, no explanation. If field missing, return null. Dates as ISO strings.
System Prompt — Bullet Rewriter
Role: Senior resume coach. Rewrite bullets using STAR method. Start with action verb.
Rules: Add quantified metrics (%, $, scale). Max 25 words. Return JSON array of strings.
Input: "Worked on backend APIs"
Output: "Architected 5 REST endpoints reducing client latency by 35%, serving 50k daily active users"

Adaptive Question Generator
Anthropic API — Dynamic MCQ
Claude generates each question fresh — never from a static bank. The prompt includes: topic domain, target difficulty (1–10), candidate's answer history, and a rule against repeating question types. Produces one MCQ with 4 options and the correct answer tagged.
claude-haiku-4-5Low latencyJSON output

Difficulty Adjuster
Anthropic API — Adaptive Logic
After each answer, Claude evaluates correctness and adjusts difficulty. Correct answer → increase difficulty by 1. Wrong answer → decrease by 1. Claude also provides a short explanation of the correct answer for learning mode.
claude-haiku-4-5Adaptive IRT-style

Voice Mode
Web Speech API + Claude
Web Speech API reads questions aloud (TTS) and transcribes spoken answers (STT). Transcribed text is sent to Claude for evaluation. Enables hands-free aptitude testing that simulates real verbal screening.

HR Persona Engine
Anthropic API — Role Play
Claude is given a system prompt that defines it as a friendly but rigorous HR interviewer at a specific company type (startup, MNC, product company). It asks STAR-format behavioral questions tailored to the candidate's resume and target role.
claude-sonnet-4-5Persona prompt
⭐
STAR Evaluator
Anthropic API — Scoring
After each answer, Claude scores STAR completeness: Situation (did they set context?), Task (what was their role?), Action (specific steps taken?), Result (quantified outcome?). Returns a JSON score per component plus improvement tip.
claude-sonnet-4-54-axis STAR score
💬
Filler Word Detector
JS + Claude NLP
Web Speech API transcribes spoken answers. A JS regex pass counts filler words (um, uh, like, you know, basically). Claude then evaluates fluency, clarity, and confidence in the transcript. Combined score weighted 30% fillers, 70% content.
